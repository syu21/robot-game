# 実況 / 分析テンプレ用カテゴリ整理

最終更新日: 2026-03-26

この文書は、今後の AI実況 / 運営分析 / X投稿 / Discord話題化で使う
`課題カテゴリ` と `語りの切り口` を先に揃えておくためのメモです。

現時点ではプレイヤー向け機能ではなく、`運営用の並行タスク` として扱います。
正本の思想は `docs/GAME_DIRECTION.md`、
実装順は `docs/IMPLEMENTATION_ROADMAP.md` を参照してください。

## 1. 基本方針

- 先に整理するのは `勝ち負け` ではなく `語れる差`
- 現フェーズでは PvP を前提にしない
- まずは `周回`, `型`, `記録`, `ボス`, `展示` の話題化を優先する
- カテゴリは `世界ログ`, `ランキング`, `展示`, `陣営戦` に繋がるものを優先する

## 2. 大カテゴリ

### 2.1 周回カテゴリ
- 最速周回
- 長時間周回
- 安定周回
- 省操作周回

使いどころ:
- 今週探索数ランキング
- 周回導線改善の評価
- X投稿での「回しやすさ」アピール

### 2.2 ボスカテゴリ
- 初撃破
- 最速撃破
- 安定撃破
- 苦手ボス突破

使いどころ:
- 世界ログ
- ボス撃破演出
- 攻略共有

### 2.3 型 / 思想カテゴリ
- 耐久型
- 命中型
- 会心型
- 爆発型
- 背水型
- バランス型

使いどころ:
- 将来の型ラベル表示
- 展示文言
- 実況テンプレ

### 2.4 進化カテゴリ
- 初進化
- 進化連打
- 推し部位R化
- 進化ルート研究

使いどころ:
- 進化成功世界ログ
- 進化画面導線
- プレイヤーの進行実感

### 2.5 記録カテゴリ
- 今週最多探索
- 今週最多ボス撃破
- 連勝
- 特定層の周回記録

使いどころ:
- ランキング
- 週間まとめ
- 記録文化の土台

### 2.6 展示カテゴリ
- このロボを見てほしい
- この型を真似してほしい
- 今週の看板ロボ
- 話題ロボ
- 今週の顔

使いどころ:
- 展示画面
- MVP表示
- ホーム下段のランキング / MVP
- 記録庫の話題ロボ
- DECOR / 見栄導線

### 2.7 陣営カテゴリ
- 今週の貢献者
- 今週の陣営MVP
- 逆転要因
- 陣営ごとの傾向

使いどころ:
- 陣営戦ログ
- 週環境まとめ
- 間接競争の強化

## 3. 実況テンプレの基本骨格

実況や分析文は、次の順番で作ると使いやすい。

1. 何が起きたか
2. どの型 / どの判断が効いたか
3. 他プレイヤーが真似できるポイントは何か
4. その出来事は世界競争のどこに繋がるか

## 4. 良いテンプレ例

- `○○が layer2 を命中型で突破`
- `○○の耐久型が今週の周回王に浮上`
- `○○が HEAD をR化、進化導線が加速`
- `○○のロボ展示が話題、同型研究が進む`
- `○○が今週の顔として記録庫とMVPに残った`

## 5. 避けるテンプレ

- ただ強かっただけで終わる文
- 戦力差だけを煽る文
- PvP前提で優劣を断定する文
- 長すぎてスクショ共有に向かない文

## 6. 使い方メモ

- 新しいランキングや世界ログを考えるときは、
  まずこのカテゴリのどこに入るかを見る
- どこにも入らない施策は、
  `周回快感`, `自然競争`, `語り` のどれにも弱い可能性が高い
