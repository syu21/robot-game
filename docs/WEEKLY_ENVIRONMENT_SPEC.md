# 週間環境仕様

最終更新日: 2026-03-26

## 1. 概要
- 週単位 (`week_key`) で環境を決め、出撃体験へ軽補正を与える
- 同じ `week_key` を陣営戦集計にも利用する
- 週環境は `ホーム下段` と `/world` の見返し面で可視化する

## 2. データモデル
### 2.1 world_weekly_environment
- `week_key`
- `element`
- `mode`
- `enemy_spawn_bonus`
- `drop_bonus`
- `started_at`, `ends_at`
- `random_seed`
- `influence_ratio`
- `reason`

### 2.2 world_weekly_counters
- `week_key`
- `metric_key`
- `value`

## 3. モード定義（代表）
- `暴走`: 出現率寄り
- `活性`: ドロップ寄り
- `安定`: 変動小
- `静穏`: 低刺激

## 4. 出撃への反映
- 敵出現重み補正
- ドロップ補正
- 一部モードで連戦率やコイン挙動に影響

## 5. 表示方針
### 5.1 `/home`
- 見出し: `今週の戦況`
- 前面表示:
  - 状態名
  - 短い効果説明
  - 下段の `今週のランキング` / `今週のMVP` と並べて世界の動きを感じさせる
- 非表示（内部保持のみ）:
  - 詳細係数
  - 生カウンタ値

### 5.2 `/world`
- 前面表示:
  - `week_key`
  - 残り時間
  - 週属性と効果説明
  - 注目属性
  - 盛り上がっている出撃先
  - 今週のMVP
- MVP は `現在の出撃機体から作る小ロボ` を主役に、補助アバターと機体画像を添えて「誰が今週の顔か」を伝える

## 6. 陣営戦連携
- 同一 `week_key` で陣営ポイントを集計
- 週締め結果を `world_faction_weekly_result` に保存
- 勝利陣営は次週軽バフ対象

## 7. 運用
- 管理導線: `/admin/world`
- 再抽選・カウンタリセットは監査を残す
